## Functions and Linear Transformations

$ D -> C $
every element in D has an element in C
ie $n\in \mathbb{R} -> f(n)$
preimage -> image

D: Domain
C: Codomain

linear transformations are functions in $\mathbb{R}^n -> \mathbb{R}^m$

> a function T in $\mathbb{R}^n -> \mathbb{R}^m$
> is a linear transformation if
> 1. $T(u + v) = T(u) + T(v) \forall u, v \in  \mathbb{R}^n$
> 2. $T(cu) = cT(u) \forall u \in \mathbb{R}^n, \forall c \in \mathbb{R}$
